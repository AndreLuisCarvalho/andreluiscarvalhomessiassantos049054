## Visão Geral

Este repositório contém uma **aplicação frontend desenvolvida em React**, utilizando **Vite** como bundler e servidor de desenvolvimento.

O projeto foi criado com o objetivo de:
- Demonstrar domínio da estrutura básica de aplicações React
- Utilizar ferramentas modernas de build
- Servir como base para avaliação técnica em processos seletivos

---

## Features

- [x] Aplicação SPA com React
- [x] Bundler moderno com Vite
- [x] Hot Module Replacement (HMR)
- [x] Estrutura simples e organizada
- [x] Scripts prontos para desenvolvimento e build

## Quick Start

### 1. Pré-requisitos

Antes de iniciar, certifique-se de ter instalado:

- **Node.js** (versão 16 ou superior)
- **npm** ou **yarn**

---

### 2. Instalação

Clone o repositório:

```bash
git clone https://github.com/AndreLuisCarvalho/andreluiscarvalhomessiassantos049054.git
Acesse a pasta do projeto:

cd andreluiscarvalhomessiassantos049054
Instale as dependências:

npm install
3. Execução
Execute o servidor de desenvolvimento:

npm run dev
A aplicação estará disponível em:

http://localhost:5173
Scripts
Comando	Descrição
npm run dev	Inicia o servidor de desenvolvimento
npm run build	Gera o build de produção
npm run preview	Visualiza o build localmente
